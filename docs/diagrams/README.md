## How to Add Diagrams 
1. Create your diagram using tools like: 
   - Draw.io (free, online) 
   - Lucidchart 
   - PlantUML (text-based) 
2. Export as PNG or PDF 
3. Name files clearly: `use-case-diagram.png`, `block-diagram.png`, etc. 
4. Commit to this folder
